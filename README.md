# sandbox1

This project integrates [agency-agents](https://github.com/msitarzewski/agency-agents) — a collection of 128+ specialized AI agent personalities designed to function as a complete virtual team.

## Agents

All agents are located in `.claude/agents/` and are automatically available in Claude Code sessions.

### Divisions

| Division | Description |
|---|---|
| `design/` | UI/UX designers, brand guardians, visual storytellers, image prompt engineers |
| `engineering/` | Frontend, backend, mobile, AI, DevOps, security, rapid prototyper |
| `game-development/` | Cross-engine designers + Unity, Unreal, Godot, Roblox specialists |
| `marketing/` | Growth hacker, content creator, social media (Twitter/TikTok/Instagram/Reddit) |
| `paid-media/` | PPC strategist, search analyst, auditor, tracking specialist |
| `product/` | Sprint prioritizer, trend researcher, feedback synthesizer |
| `project-management/` | Studio producer, project shepherd, operations, experiment tracker |
| `spatial-computing/` | AR/VR/XR interface designers, Vision Pro, WebXR |
| `specialized/` | Agents orchestrator, data analytics, LSP engineer, sales/identity systems |
| `strategy/` | Strategic planning agents |
| `support/` | Customer service, analytics reporter, finance tracker, compliance |
| `testing/` | QA, performance benchmarker, API tester, accessibility auditor |

## Usage

Agents are available automatically within Claude Code. Reference an agent by its role to activate its specialized behavior, or use the agents orchestrator to coordinate multi-agent workflows.

### Example team compositions

- **Startup MVP:** Frontend Developer + Backend Architect + Growth Hacker + Rapid Prototyper + Reality Checker
- **Marketing campaign:** Content Creator + Social Media Strategist + Paid Media agents + Analytics Reporter
- **Enterprise development:** Senior Developers + Security Engineers + DevOps + comprehensive testing division
